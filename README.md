# Mailer

## Git Clone

~~~
git clone https://github.com/elkulo/Mailer.git Mailer
cd Mailer
npm install
npm develop
~~~

env を .env にリネーム

## Git First Install

~~~
git init
git add README.md 
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/elkulo/Mailer.git
git push -u origin main
git remote set-head origin main
git add --all
git commit -m "Mailer"
~~~
